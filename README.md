# 🌌 Deepika Medisetti - Data Scientist Portfolio

A sleek, interactive, and fully responsive personal portfolio website designed to showcase my skills, experience, and projects as a Data Scientist and Data Analyst.

Built with modern web technologies, this portfolio features a glassmorphism UI, a dynamic animated background, and seamless dark/light mode toggling.

## 🚀 Features

* **Interactive Canvas Background:** A custom-built, animated network node background that responds to window resizing.
* **Glassmorphism UI:** Modern, translucent card designs with elegant hover effects.
* **Theme Toggling:** Seamless switching between Dark and Light themes with local storage persistence.
* **Scroll Animations:** Smooth reveal animations as you scroll down the page using the Intersection Observer API.
* **Fully Responsive:** Optimized for desktops, tablets, and mobile devices with a custom hamburger menu.

## 🛠️ Tech Stack (Portfolio Build)

* **HTML5:** Semantic structure and layout.
* **CSS3:** Custom variables, CSS Grid/Flexbox, glassmorphism, and media queries.
* **Vanilla JavaScript:** DOM manipulation, canvas rendering, theme toggling, and scroll observers.
* **Fonts:** 'Dancing Script' and 'Nunito' (via Google Fonts).
